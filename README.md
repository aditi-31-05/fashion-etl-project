📌 Project Overview:-

-This project focuses on building an ETL pipeline to clean and analyze a messy fashion dataset. The data was processed using Python, stored in MySQL, and visualized using Power BI to generate meaningful business insights.

🎯 Objective:-

-To transform raw, unstructured fashion data into a clean and analyzable format and derive insights related to pricing, ratings, brands, and product trends.

🛠️ Tools & Technologies:-

-Python (Pandas)
-MySQL
-Power BI

🔄 ETL Process:-

-Extract: Loaded raw CSV dataset
-Transform: Cleaned data (handled nulls, removed noise, standardized formats, created new features)
-Load: Stored cleaned data into MySQL database

📊 Key Insights:-

-Medium and high price categories dominate the dataset
-Higher price does not always lead to higher ratings
-Some brands consistently perform better in ratings
-Colour preferences show strong dominance of specific shades
